# store-front

A store-front website that helps small businesses to show case their goods and services online.

# Requirements
1. MongoDB or MongoAtlas
2. Node Js
3. Browser
4. git bash


## usage for the client
``` bash
cd client && npm install
npm start
```
## to run the server
```bash
#
cd server
npm install
npm run server 
```
